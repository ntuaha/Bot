### 功能類別

1. Web Plugins

有兩種類別Plugins可以放在網站(desktop web and mobile web)．其中一種是Send To Messenger可以帶一個訊息送到聊天室, 另外一種叫做Message Us,則是開啟與我們的聊天室．在Desktop web, 是開啟Messenger.com. 而在Mobile web則會嘗試開啟Messenger native app.
